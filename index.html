End of arm Camera Mount

Designed a camera mount to mount onto either side of the end-of-arm tool and monitor pick and place. The goal was to monitor the pick and place actions to collect unobstructed and clear images of instances of failed action and use the images for image based retry after failure.

Hardware Test Rig

Six-degree-of-freedom stepper motor operated test rig for both pick and place to stress test computer vision system and test edge cases.

motion control system

Tongue operated assistive device made of bio-safe materials to help people with locomotor disorders using touch sensors to actuate a wheelchair.

Tool Centre Point Camera

Camera mount designed to have an unobstructed view to inspect and correct the Tool Centre Point (TCP) of the robot's end-of-arm tool.

Pressure Sensors

Fabricated and packaged sensors utilizing low-pressure chemical vapor deposited (LPCVD) amorphous silicon nitride (SiNx) on single crystal Si wafers within a controlled environment of Class 100 and Class 1000 cleanrooms.

Animatronic Arm

Designed an animatronic twin control system that replicates a flex sensor-based wearable glove, capable of detecting even slight changes in resistance from subtle finger movements.

Gait Analysis of a Swimming flagellum

Implemented a code on PyElastica to compute the mechanics of soft filaments using Cosserat rod theory and 2nd-order Position Verlet time-stepping, to construct a near-realistic model of a swimming flagellum and optimized the gait using a version of CMAes to find the maximal forward velocity, converging within 4000 generations.

SPV Panel Cooling system

Manufactured a cooling system for Solar Photovoltaic panels using phase change materials (PCMs) and tested the framework using a 5 W panel, achieving an increase in conversion efficiency by around 16.5%.
